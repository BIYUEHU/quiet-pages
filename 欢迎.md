是你的新*仓库*。

写点笔记，[[创建链接]]，或者试一试 [导入器](https://help.obsidian.md/Plugins/Importer) 插件!

当你准备好了，就将该笔记文件删除，使这个仓库为你所用。

## Structure

```
writing-vault/
 ├─ journal/        ← 记录、日记
 ├─ fiction/        ← 小说创作
 ├─ essays/         ← 随笔（读书/玩游戏的感想）
 ├─ summaries/      ← 总结
 ├─ tech/           ← 技术文章、学习总结
 ├─ notes/          ← 笔记（读书笔记、课堂笔记）
 ├─ reviews/        ← 书评、观影感想
 ├─ misc/           ← 难以分类的杂项
 └─ lexicon/        ← 词库（Various Complements 用）

```
