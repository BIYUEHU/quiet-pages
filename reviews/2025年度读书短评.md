- 按时间顺序升序排列，不代表排名先后（本文也无意排名）
- 类型涉及较多，但大成还是文学类
- 默认只点评阅读完毕的书籍，且为实体书
- `"严肃文学"` 分类包含小说、散文、随笔等各种形式
- 每项符合以下类型定义：

```typescript
interface 书评单例 {
  译名: Maybe<String>
  原名: String
  作者: String
  译者: Maybe<String>
  国家: Country
  类型: 严肃文学 | 历史 | 技术 | 通俗文学 | 轻小说 | 哲学 = 严肃文学
  引用: Maybe<[String]>
  实体书: 是 | 否 = 是
  描述: Text
}
```
